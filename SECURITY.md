# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Elaborate, please report it responsibly.

**Do not open a public issue.**

Instead, [create a private security advisory](../../security/advisories/new) on this repository.

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Fix or mitigation**: Depends on severity, but targeting 30 days for critical issues

## Scope

This policy covers the Elaborate source code and its published npm package. It does not cover third-party dependencies (report those upstream).
